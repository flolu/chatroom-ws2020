# Requirements

Linux, Node.Js, Bazel, Docker Compose, Yarn

<br>

# Commands

**Setup**

```
yarn install && yarn build
```

**User Client**

```
yarn user-client:dev
yarn user-client
```

**Server Client**

```
yarn server-client:dev
yarn server-client
```
